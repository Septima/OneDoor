Installation ​

OneDoor tilbydes som as-a-service eller som on-premise løsning.

As-a-service ​

Ingen installation kræves. Tilmeld dig og få adgang inden for 30 dage.

Prøv OneDoor gratis

On-premise ​

OneDoor kan installeres på en server i jeres eget IT-miljø.

Forudsætninger ​

  • Node.js 18+
  • En server med adgang til internettet

Installer ​

bash
npm install @septima/onedoor-server

Start ​

bash
npm start
